# Animalitos

### Tecnologías

+ PHP (5.3+)
+ SQLite3
+ Composer
+ Bower
+ NodeJS - NPM - Gulp

### Instalación - Despliegue

 	$ composer install

### Para recargar el autoload de clases:

 	$ composer dump-autoload -o && npm install && bower install 

### Generar 'dist'
	
	$ gulp layout && grub home
